# 简单的 MDX 演示

这是一篇使用 MDX 格式编写的文章。

## 普通 Markdown 语法

MDX 完全支持所有标准的 Markdown 语法：

### 列表
- 项目 1
- 项目 2
- 项目 3

### 代码块

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet("MDX");
```

## 基本 HTML 元素

MDX 支持在 Markdown 中使用 HTML：

<div style="background: #f0f0f0; padding: 20px; border-radius: 8px; text-align: center; margin: 20px 0;">
  <h3>这是一个简单的盒子</h3>
  <p>使用基本 HTML 创建的样式化内容</p>
</div>

<div style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); color: white; padding: 25px; border-radius: 12px; text-align: center; margin: 25px 0;">
  <h2>美丽的渐变卡片</h2>
  <p>使用内联 CSS 样式创建</p>
</div>

## 结论

MDX 结合了 Markdown 的简洁性和 HTML 的强大功能，非常适合创建丰富的内容。

**主要优势：**
1. 保持 Markdown 的简洁性
2. 支持 HTML 元素
3. 可以创建样式化的内容
4. 易于维护和扩展
