### 命令模板（需要替换用户名）:

```powershell
# 1. 移除旧的远程仓库地址
git remote remove origin

# 2. 添加您的私有仓库地址
git remote add origin https://github.com/您的用户名/my-astro-blog.git

# 3. 推送到您的私有仓库
git push -u origin main
```

## 🔒 确保仓库设置为私有

创建仓库时，请确保：
- ✅ 选择 **Private** 
- ✅ 不勾选 README、.gitignore、license
- ✅ 仓库名建议：`my-astro-blog`
